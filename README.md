##Description
This photography portfolio website is built with HTML and CSS. It is designed to showcase a collection of photographs in a clean, responsive, and visually appealing manner. The website includes a header with navigation, a hero section a gallery of photos,an about section and a contact form.

##live site
the project is deployed on github pages and can be accessed at:
https://randyombiro8-pixel.github.io/photographywebsite/

##setup Instructions 
1.clone the repository
2.open the project folder
3.open 'index.html' in a web browser
